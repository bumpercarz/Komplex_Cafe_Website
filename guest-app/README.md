# Komplex Cafe Online Ordering System
In fulfillment for Software Engineering II in the College of Computing and Information Sciences, University of Santo Tomas, Manila


## Do before running
Run these commands in the terminal to ensure that the website works as intended:

`npm install`
to install main software packages and manages dependencies

`npm install react-router-dom`
for routing and navigation

`npm install react-icons`
for react icons to display properly in the website

`npm install firebase`
to implement firebase


## How to run
Use the command in the terminal:
`npm run dev -- --host`

and `Ctrl + C` when closing the application
